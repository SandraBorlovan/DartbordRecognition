## How to build and run:

Before build: change path in Makefile to correct directory

To build: 
	- open a terminal in the containing folder;
	- type 'make' and hit enter, this will generate the executable;

To Run:
	- write './detect ' + 'dart' + no. of image + extension and hit enter;
	- e.g. for image 0: './detect dart0.jpg'.

Note: Images have to be in the same directory as the executable.
